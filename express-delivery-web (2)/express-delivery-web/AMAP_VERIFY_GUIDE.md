# 高德开放平台 - 个体工商户验证指南

## ✅ 推荐选择：企业开发者

**作为个体工商户，你应该选择"企业开发者"，原因如下：**

### 1. 符合资质要求
- ✅ 个体工商户拥有营业执照
- ✅ 可以使用营业执照进行企业认证
- ✅ 高德平台支持个体工商户使用营业执照认证为企业开发者

### 2. 权益优势对比

| 项目 | 企业开发者 | 个人开发者 |
|------|----------|----------|
| **免费额度** | 提升500% | 基础额度 |
| **开票服务** | 企业专属开票 | 仅支持个人开票 |
| **流量包** | 支持按量计费 | 不支持额外购买 |
| **商业用途** | ✅ 支持 | ❌ 仅限学习使用 |
| **认证材料** | 营业执照+法人身份证 | 身份证 |

### 3. 需要准备的材料

选择"企业开发者"后，需要准备：

1. **营业执照**
   - 个体工商户营业执照照片
   - 确保清晰可见，信息完整

2. **法人身份证**
   - 法人身份证正反面照片
   - 与营业执照上的法人信息一致

3. **其他可能需要的材料**
   - 联系人信息
   - 联系电话
   - 邮箱地址

## 📝 验证流程

### 第一步：选择"企业开发者"
点击页面上的 **"企业开发者"** 选项

### 第二步：填写企业信息
- 企业名称：填写营业执照上的名称
- 统一社会信用代码：营业执照上的18位代码
- 法人姓名：与营业执照一致
- 联系人信息：填写你的联系方式

### 第三步：上传认证材料
- 上传营业执照照片
- 上传法人身份证正反面

### 第四步：等待审核
- 审核时间通常为1-3个工作日
- 审核通过后即可创建应用和API Key

## ⚠️ 注意事项

1. **营业执照类型**
   - 确保营业执照类型为"个体工商户"
   - 不是"有限责任公司"等其他类型

2. **信息一致性**
   - 所有填写的信息必须与营业执照一致
   - 法人姓名必须与身份证一致

3. **照片清晰度**
   - 营业执照和身份证照片必须清晰
   - 建议使用扫描件或高清照片

4. **如果审核不通过**
   - 可以联系高德客服咨询
   - 或者先选择"个人开发者"进行测试（但功能受限）

## 🚀 下一步操作

完成企业开发者认证后：
1. 进入"应用管理"创建应用
2. 添加Web端(JS API)的Key
3. 配置IP白名单
4. 获取API Key并配置到项目中
