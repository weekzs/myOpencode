# 快速开始指南

## 方法1：使用批处理文件（推荐）

直接双击运行 `运行脚本.bat` 文件，脚本会自动：
1. 检查Python是否安装
2. 检查并安装依赖包
3. 运行爬虫脚本

## 方法2：手动运行

### 步骤1：安装依赖

打开命令提示符（CMD）或PowerShell，进入项目目录，运行：

```bash
pip install -r requirements.txt
```

### 步骤2：运行脚本

```bash
# 方式1：运行示例脚本（会爬取所有章节）
python example_usage.py

# 方式2：直接运行主脚本
python comic_crawler.py
```

## 方法3：在Python中运行

如果你想先测试一下，可以修改 `example_usage.py`，取消注释 `example2()` 来只爬取前10章：

```python
if __name__ == "__main__":
    # example1()  # 注释掉
    example2()  # 取消注释，只爬取前10章用于测试
```

## 注意事项

1. **首次运行建议先测试**：修改 `example_usage.py` 使用 `example2()` 只爬取前10章
2. **网络连接**：确保网络连接正常，可以访问目标网站
3. **存储空间**：确保有足够的磁盘空间（所有章节可能需要几GB空间）
4. **运行时间**：爬取所有章节可能需要较长时间，请耐心等待

## 输出位置

下载的图片会保存在：
- `images/` 目录下，按章节组织
- 每个章节会创建一个独立的文件夹，格式为：`0001_章节名称/`
- 图片文件命名格式：`0001.jpg`, `0002.jpg` 等

## 常见问题

### Q: 提示找不到模块？
A: 运行 `pip install -r requirements.txt` 安装依赖

### Q: 下载失败？
A: 检查网络连接，重新运行脚本会自动跳过已下载的图片

### Q: 想只爬取部分章节？
A: 修改 `example_usage.py` 中的参数，例如：
```python
crawler.crawl_all(start_chapter=0, end_chapter=10)  # 只爬取前10章
```
